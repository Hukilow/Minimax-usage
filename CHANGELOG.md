## [0.1.0] — 2026-06-25

### Added
- Status bar (5h + weekly) with color tiers and reset countdowns.
- Sidebar TreeView with per-model rows.
- Detail webview dashboard with uPlot history chart.
- Commands: Set / Clear API Key, Refresh, Open Dashboard, Open Billing, Toggle Status Bar, Toggle Sidebar.
- Settings: refresh interval, status bar mode, thresholds, history size, sidebar visibility, debug logging.
- SecretStorage-backed API key handling.
- MIT license, CI workflow, Copilot customizations for contributors.
